# Assets

Structured copy of all brand, background, icon, and social assets used in the EvalAI portal.

## Folder Structure

```
assets/
├── brand/
│   ├── logos/
│   │   ├── sss-logo-horizontal-white.png   — Horizontal SSS logo (white) — used in PortalHeader
│   │   ├── sss-logo-horizontal-white-orig.png — Original source file
│   │   ├── sss-logo-vertical-white.png     — Vertical SSS logo (white)
│   │   └── logo-white.png                  — Generic white logo variant
│   ├── marks/
│   │   ├── sss-mark-white.png              — SSS bolt mark (white) — used on dark login card
│   │   ├── sss-mark-black.png              — SSS bolt mark (black) — used on light login card
│   │   ├── sss-mark-white-orig.png         — Original source file
│   │   └── sss-mark-black-orig.png         — Original source file
│   └── wordmarks/
│       ├── sss-wordmark-white.png          — Text wordmark (white)
│       └── sss-wordmark-white-orig.png     — Original source file
├── backgrounds/
│   └── bg-power.png                        — Power infrastructure photo — login page background
├── icons/
│   └── favicon.svg                         — Browser tab icon
└── social/
    └── opengraph.jpg                       — OG image for social/link previews
```

## Notes
- Files ending in `-orig` are the original uploaded source files with original filenames preserved.
- The live app references assets from `artifacts/evaluations-portal/public/` — this folder is a structured reference copy.
